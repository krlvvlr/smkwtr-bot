# WaterBot

Telegram bot for automated content protection and watermarking.

## Project Goal

WaterBot helps Telegram channel owners and content creators protect their media by automatically applying custom watermarks to photos and videos.

## Features

- Photo watermarking
- Video watermarking
- Batch media processing
- Telegram channel integration
- Trial access system
- Subscription logic
- Admin panel
- Usage statistics

## Technology Stack

- Python
- Aiogram 3
- Pillow
- MoviePy
- SQLite

## Use Cases

- Telegram channels
- Content creators
- Premium communities
- Digital content protection
